# building

in case you don't trust my silly binaries, you can build it yourself!

```bash
# clone the repo
git clone https://github.com/xtrm-en/barman
cd barman

# run gradle to build the mod
chmod +x gradlew
./gradlew build

# the mod will be in build/libs
cd build/libs
ls -l

# yipee
```
